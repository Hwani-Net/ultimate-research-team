# 🦅 Ultimate Research Team 마스터 가이드 (v11.1)

본 시스템은 최첨단 리서치 아키텍처인 **'5-Agent Deep Strategy' 체제**를 기반으로 합니다. 단순한 질문보다 '전략적 지시'를 내릴 때 최고의 성능을 발휘합니다.

## 🎯 최고의 결과를 얻는 '고급 프롬프팅' 전략

에이전트들에게 페르소나와 제약을 줄 때 다음 5가지 요소를 포함하세요.

### 1. 명확한 페르소나 부여 (Who)
단순히 "조사해줘"가 아니라, 에이전트들의 역할을 규정해 주세요.
*   *예시: "너희는 포춘 500대 기업의 전략 컨설팅 팀이야. 아주 차갑고 객관적인 시선으로 리스크를 파헤쳐줘."*

### 2. 적대적 검증 요청 (Adversarial Thinking)
**Chief Skeptic (비평가)**가 더 날카로워지도록 지시하세요.
*   *예시: "비평가는 리서처의 보고서에서 최소 3가지의 논리적 허점을 찾아내고, 반대되는 데이터가 있는지 반드시 확인해."*

### 3. 수치 데이터 집중 (Quant-X)
**Data Analyst (분석가)**에게 구체적인 지표를 요구하세요.
*   *예시: "2024-2026년의 CAGR(연평균 성장률)과 시장 점유율(Market Share) 데이터를 반드시 표로 정리해."*

### 4. 비즈니스 실행 계획 (Execution)
**Strategist (전략가)**에게 돈이 되는 모델을 요구하세요.
*   *예시: "이 기술을 도입했을 때의 ROI(투자타당성) 분석과 1년차 실행 로드맵(Gantt)을 그려줘."*

### 5. 📸 멀티모달 비전 활용
이미지를 업로드하면 **'시각 지능(Visual Intelligence)'**이 활성화됩니다.
*   *활용법: 제품 사진, 차트 스크린샷, 현장 사진 등을 업로드하고 "이 이미지의 시장 반응을 분석해줘"라고 지시하세요.*

## 🛠️ 시스템 운영 꿀팁 (Soulless Mode v11.1)

- **✨ Magic Prompt (프롬프트 분기)**: 
  - **3인 모드**: 간결한 요약과 최신 팩트 위주의 지시서 생성.
  - **5인 모드**: 투자 방어, 수익 모델링, 규제 검토가 포함된 심층 전략 지시서 생성.
- **⚖️ A/B Testing (신규)**: 'Compare Modes'를 체크하면 3인/5인 모드를 동시 수행하여, 속도와 깊이의 차이를 한 눈에 비교하는 보고서를 생성합니다.
- **💰 Cost Estimator**: 사이드바에서 현재 모드의 예상 비용($0.02 vs $0.15)을 확인하고 경제적으로 운용하세요.
- **🛡️ Investment Defense**: 5인 모드 선택 시 'Risk Hedge Report'와 'K-Compliance(한국형 규제)' 검토가 자동으로 수행됩니다.
- **한글 보고서**: 모든 결과물은 분석 후 하단에 **상세한 한글 요약 및 번역**이 포함됩니다. (v11.1: 한국어 응답 원칙 강화)

## ⚠️ 할당량(Quota) 및 에러 관리
- **429 에러**: 구글 API의 분당 요청 한도를 초과할 때 발생합니다. 잠시 기다렸다가 다시 [Start Research]를 눌러주세요.
- **404 에러**: 모델명이 유효하지 않을 때 발생합니다. 현재 시스템은 2026년 표준인 `Gemini 2.5/3.0` 하이브리드 시리즈에 최적화되어 있습니다.
